    8. Exit: `const hasError = findings.some(f => f.severity === 'error')`, `const hasWarn = findings.some(f => f.severity === 'warn')`, `if (hasError) process.exit(2); if (hasWarn) process.exit(1); process.exit(0)`

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-02-10 | Tampering | CLI --staged flag passed to git diff | mitigate | git args are passed as array (not shell string interpolation); T-1-06 from Phase 1 RESEARCH.md already verified this pattern in runGitDiff |
